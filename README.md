# appdevelopment

YourOwnJournal is a .NET MAUI app. The solution file is `appdev2.sln` and the main project lives under `YourOwnJournal/`.

## Requirements
- .NET SDK with MAUI workloads for your target platform.

## Build
- Open `appdev2.sln` in Visual Studio (or run `dotnet build` for a specific target framework).

## Project layout
- `YourOwnJournal/` application source
- `YourOwnJournal/Docs/` documentation and schema notes
